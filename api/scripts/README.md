# סקריפטים

## טעינת ספר "תחשיבי נזק" למאגר העוזר

הסקריפט `ingestBookFromFolder.ts` קורא את כל קבצי ה-PDF מתיקיית הספר (ברירת מחדל: `content/books/תחשיבי-נזק` ביחס לשורש הפרויקט), מחלץ מהם טקסט ויוצר רשומת מסמך לכל פרק במאגר. כך העוזר (Chatbot) יכול לחפש בתוכן הספר.

**דרישות:** מסד נתונים פעיל (`.env` עם `DATABASE_URL` בתיקיית `api`).

**הרצה מתוך תיקיית `api`:**

```bash
cd api
npm run ingest-book
```

**תיקייה אחרת:** אפשר להגדיר משתנה סביבה:

```bash
set BOOK_FOLDER=C:\path\to\your\book\folder
npm run ingest-book
```

אם מסמך עם אותו כותרת וספר כבר קיים – הסקריפט ידלג עליו (לא יכפיל).
